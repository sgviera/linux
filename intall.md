# Install stuff

## From Ubuntu repository (apt)

### Search for update

```
sudo apt update
```

### Upgrade the version of all installed software

```
sudo apt upgrade
```

### Search for specific software

```
sudo apt search telegram
```

### Install specific package

```
sudo apt install git
```

## From .deb files (dpkg)

### Install

```
dpkg -i filename.deb
```
