# Filesystem

Here there is a summary of all my learnings on how to explore a file system with a terminal.

## Useful Commands

### Show Working Directory

```bash
pwd
```

### Change Directory

```bash
cd
```

### Show Directory Contents

```bash
ld
```

### Show File Contents

```bash
cat
```

### Create File

```bash
touch
```

### Create Directory

```bash
mkdir
```

### Copy

```bash
cp
```

### Remove

```bash
rm
```

### Move

```bash
mv
```

### Edit File in Terminal

```bash
nano
```
